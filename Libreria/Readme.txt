HOW TO INSTALL 

Place this folder inside Documents/MATLAB folder, then run SkywardConfigureScript.m. 
When the automatic procedure is finished, you can find the Library in the Simulink Browser.

Current Sensor Supported : 

BMP085
MPU6050
HMC5883L
HC-SR04

If you want more sensors support let me know.

Current OS Supported : only Windows 7/8 64bit
MATLAB Version Supported : R2014a
